<h2> Como usar </h2>

   ```
   - Clone this repository:
   $ git clone https://github.com/e-medeiros1/e_shop 
   - Enter in directory:
   $ cd e_shop
   - For install dependencies:
   $ flutter pub get
   - Run the app: 
   $ flutter run
   ```
